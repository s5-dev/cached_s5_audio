# cached_s5_image

A simple cached audio widget for S5 CID's

![Demo](https://raw.githubusercontent.com/s5-dev/cached_s5_audio/main/static/demo.gif)

### Usage

See [example](example/lib/main.dart).

This is a library built on [s5](https://pub.dev/packages/s5). See there for more details.

Basic Usage:

```dart
CachedS5Audio(
    cid: cid!, // String representation of CID
    s5: s5!, // S5 singleton
)
```
